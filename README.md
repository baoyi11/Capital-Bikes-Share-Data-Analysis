# Capital Bikeshare Data Analysis Dashboard

A comprehensive Streamlit web application for analyzing Capital Bikeshare trip data with interactive visualizations and data storytelling.

## Features

- **Interactive Dashboard**: Filter by date, user type, bike type, and time of day
- **Multiple Visualization Types**: 
  - Time series charts
  - Geographic maps and heatmaps
  - Comparative charts and bubble charts
  - Distribution analysis and box plots
- **Data Storytelling**: Structured narrative from problem statement to insights and recommendations
- **Performance Optimized**: Cached data processing for fast loading
- **Responsive Design**: Works on desktop and mobile devices
